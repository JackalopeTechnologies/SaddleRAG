# Contributing to SaddleRAG

Thank you for your interest in contributing to SaddleRAG.

## License of Your Contribution

SaddleRAG is released under the [MIT License](./LICENSE). By submitting a pull request, you agree that your contribution will be licensed under the same MIT terms. No separate Contributor License Agreement is required.

## How to Contribute

1. Open a pull request with your proposed change.
2. The maintainer will review and either merge, request changes, or close with feedback.
3. Discussion happens in the pull request itself — keep commit messages focused on what changed and why.

## Bug Reports and Feature Requests

The GitHub issue tracker is for bug reports and feature requests. Please include enough information to reproduce a bug (versions, OS, steps, expected vs actual behavior) so the issue can be addressed.
